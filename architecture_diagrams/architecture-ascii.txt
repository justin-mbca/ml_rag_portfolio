================================================================================
  Data Flow:
  1. Documents loaded via Prefect ingest_flow
  2. Ingestion-service chunks, embeds, indexes to ChromaDB
  3. Graph-service extracts entities/relations to Neo4j
  4. User queries API Gateway → Agent Service
  5. RetrievalAgent calls RAG Service (hybrid VectorRAG + GraphRAG)
  6. ComplianceAgent + RiskAnalysisAgent enrich the answer
  7. Structured response returned to client
================================================================================
